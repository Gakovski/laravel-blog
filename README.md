
# Laravel 9 - Blog 

Lab exercise for the faculty course Implementation of Free and Open Source Systems.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Gakovski/laravel-blog
```

Go to the project directory

```bash
  cd laravel-blog
```

```bash
  ./vendor/bin/sail up -d
```
